Section 1: Analogue and Digital Signals

Electronic circuits process electrical signals. A signal is a voltage that carries information — it tells the circuit when to switch on or off, or what value something is. There are two fundamentally different types of signal:

Analogue signals

An analogue signal can be any voltage value between a minimum and maximum. It changes smoothly and continuously, like a dimmer switch or a volume control. A good example is the light level outside — it gradually gets brighter from dawn, reaches a peak at midday, and slowly fades to dark.

Digital signals

A digital signal can only be one of two values — ON or OFF. In electronics, these are represented as:

Logic 1 = ON = HIGH = approximately +5V     Logic 0 = OFF = LOW = 0V

A simple switch is a digital input — it is either open (0) or closed (1). There is no in-between. All computers, smartphones and digital systems work entirely with 1s and 0s.

Section 2: Logic Gates

A logic gate is an electronic component that takes one or more digital inputs and produces a single digital output. The output depends entirely on the combination of inputs — and that relationship is described precisely using a truth table and a Boolean expression.

Logic gates are the building blocks of all digital systems — from simple alarm circuits to the billions of transistors inside a modern CPU.

For N5 Engineering Science you must know: NOT, AND and OR gates — their symbols, truth tables and Boolean expressions. You should also be familiar with NAND and NOR as combinations of these.

The NOT gate (Inverter)

The NOT gate has one input and one output. It simply inverts the input — if the input is 1, the output is 0, and vice versa. The small circle (bubble) on the output of the symbol represents inversion.

TRUTH TABLE

Input AOutput Z
01
10

BOOLEAN EXPRESSION

Z = A
"Z equals NOT A" — the bar above A means NOT

Real-world example: A NOT gate is used in a burglar alarm where a pressure pad is normally pressed (logic 1 = safe). When someone steps off the pad it goes to 0, and the NOT gate inverts this to switch the alarm ON (logic 1).

The AND gate

The AND gate has two (or more) inputs and one output. The output is only logic 1 when ALL inputs are logic 1 simultaneously. If any input is 0, the output is 0.

TRUTH TABLE

Input AInput BOutput Z
000
010
100
111

BOOLEAN EXPRESSION

Z = A.B
"Z equals A AND B" — the dot means AND

Real-world example: A safety press on a machine — both the operator's left hand button (A) AND right hand button (B) must be pressed before the machine operates. This prevents the operator from accidentally putting a hand under the press.

The OR gate

The OR gate has two (or more) inputs and one output. The output is logic 1 when ANY input is logic 1. The output is only 0 when all inputs are 0.

Input AInput BOutput Z

101
111

BOOLEAN EXPRESSION

Z = A+B
"Z equals A OR B" — the + means OR

Real-world example: A doorbell that can be triggered by the front door button (A) OR the back door button (B) — pressing either one rings the bell.

NAND and NOR gates

NAND and NOR gates are simply AND and OR gates with a NOT (inverter) on the output. The bubble on the output symbol indicates inversion.

Section 3: Boolean Algebra

Boolean algebra was developed by the British mathematician George Boole in the 1850s. It gives us a precise mathematical language to describe the behaviour of logic circuits. Today it is used by every electronic engineer, computer scientist and software developer.

Boolean notation summary

SymbolMeaningExampleEnglish equivalent
ANOT AZ = AZ is ON when A is OFF
.ANDZ = A.BZ is ON when A AND B are ON
+ORZ = A+BZ is ON when A OR B (or both) are ON
A.BNANDZ = A.BZ is ON unless both A AND B are ON
A+BNORZ = A+BZ is ON only when both A and B are OFF

Writing a Boolean expression from a circuit

Work through the circuit left to right, writing an expression after each gate. Bracket sub-expressions where needed.

Worked Example — Boolean expression from circuit
1
Label the output of the OR gate (B,C): B+C
2
The AND gate takes A and (B+C) as inputs: Z = A.(B+C)
3
Complete expression: Z = A.(B+C) — "Z is ON when A is ON AND at least one of B or C is ON"

Section 4: Combinational Logic Circuits

When two or more logic gates are connected together to perform a specific task, the result is called a combinational logic circuit. The output depends on the combination of all current inputs.

How many rows in a truth table?

The number of rows in a truth table doubles with each additional input:

Number of inputsNumber of rows
12
24
38
416
The inputs are always listed in binary counting order — start from all 0s and count up in binary to all 1s. This ensures every combination is covered exactly once.

Method for completing a truth table from a circuit

Label the output of each intermediate gate with a letter (D, E, F etc.). Then fill in one column at a time, working left to right through the circuit.

Worked Example — Completing a truth table step by step

Circuit: A and B feed an OR gate (output D). C feeds a NOT gate (output E). D and E feed an AND gate (output Z).

1
Column D (OR gate, inputs A and B): D = 1 whenever A OR B = 1.
2
Column E (NOT gate, input C): E is the inverse of C.
3
Column Z (AND gate, inputs D and E): Z = 1 only when D AND E are both 1.

Section 5: Boolean Expressions from Truth Tables

Sometimes you will be given a truth table and asked to write the Boolean expression and draw the circuit. The method is straightforward — focus only on the rows where the output is logic 1.

Method

  1. Look at the truth table and highlight every row where Z = 1
  2. For each highlighted row, write an AND expression for that combination of inputs. If an input is 0, write its complement (with a bar): e.g. A. If it is 1, write it normally: A.
  3. Connect all the row expressions with OR (+) to get the full Boolean expression
  4. Draw the logic circuit from the expression

Section 6: Designing Circuits from a Specification

The most common engineering application of digital logic is designing a circuit to meet a given written specification. Read the specification carefully — the key words AND, OR and NOT tell you which gates are needed.

Method

  1. Identify all the inputs from the specification — name them A, B, C etc.
  2. Identify the output — name it Z
  3. Read the specification and identify the logic: look for AND, OR, NOT
  4. Write the Boolean expression
  5. Draw the logic circuit
  6. Draw up the truth table (all input combinations)
  7. Verify using the simulator
Worked Example — Museum burglar alarm

"A burglar alarm should sound if the master switch is on AND either a light beam is broken OR a pressure pad is triggered."

1
Inputs: Master switch (M), Light sensor (L), Pressure pad (P). Output: Alarm bell (B)
2
Logic: The light sensor triggers when the beam is broken — so it goes to 0 when a burglar passes. We need NOT L. The pressure pad goes to 1 when stood on.
3
Boolean: B = M.(L + P)
4
Circuit: L feeds a NOT gate. NOT-L and P feed an OR gate. The OR output and M feed an AND gate. The AND output is B.

Section 7: Logic Gate Integrated Circuits (ICs)

In practice, logic gates are not individual components — they are built into integrated circuits (ICs), also called chips. Each IC contains several logic gates in a single small package.

The chips used in school are TTL (Transistor-Transistor Logic) chips, which require a stable +5V supply. The chip number always starts with 74:

IC numberContentsDescription
74046 × NOT gatesHex inverter — six independent NOT gates
74084 × 2-input AND gatesQuad 2-input AND
74324 × 2-input OR gatesQuad 2-input OR
74004 × 2-input NAND gatesQuad 2-input NAND
74024 × 2-input NOR gatesQuad 2-input NOR
74864 × 2-input XOR gatesQuad 2-input Exclusive OR
Pin 14 connects to +5V (Vcc). Pin 7 connects to 0V (GND). These power connections must always be made or the IC will not work.

Section 8: Real-World Applications of Logic Gates

Logic gates are found in almost every electronic system. Understanding how they work allows engineers to design control systems, safety systems, automation and more.

🔒 Security systems
AND gates ensure multiple conditions must be met (e.g. correct PIN entered AND correct card present) before access is granted.
🏭 Industrial safety
Two-hand control presses require AND logic — both buttons must be pressed simultaneously before the press activates, keeping hands safe.
🚗 Car systems
Airbags deploy using complex combinational logic — multiple sensors (crash severity, speed, seatbelt status) feed AND/OR combinations.
🌡️ Environmental control
NOT gates invert thermistor signals so heating switches on when temperature is LOW (not high), using NOT logic for active-low sensors.
💊 Medical equipment
Ventilators use logic gates to decide when to deliver a breath — combinations of patient effort sensors and timing signals feed AND/OR gates.
🤖 Robotics
Obstacle avoidance — OR gate triggers braking if left sensor OR right sensor detects an obstacle. AND gates confirm both sensors clear before resuming.
